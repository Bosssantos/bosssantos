Introdução

O que é HTML?

HTML (HyperText Markup Language) é a linguagem padrão para criar páginas web. Ela usa tags para estruturar o conteúdo.

Elementos HTML Comuns

Títulos e Parágrafos

As tags h1 até h6 definem títulos, e a tag p define parágrafos.

Listas

Você pode criar listas ordenadas (<ol>) e não ordenadas (<ul>):

  • Primeiro item
  • Segundo item
  • Terceiro item

Links e Imagens

Tags importantes para navegação e mídia:

  • <a href="url"> - para criar hiperlinks
  • <img src="url" alt="texto"> - para inserir imagens

Sobre este Site

Este é um site de exemplo criado com uma estrutura HTML básica e bem organizada. Ele demonstra o uso correto de tags semânticas e boas práticas de desenvolvimento web.
